To run script, simply download or clone

- `cd cytix`
- `npm install`
- `touch .env`
- paste the following into the newly created .env file
    - API_URL=https://api.rig.cytix.io/api
    - USERNAME=admin
    - PASSWORD=password
    - SCAN_URL=http://example.cytix.io

- `npx ts-node mainscanner.ts`
